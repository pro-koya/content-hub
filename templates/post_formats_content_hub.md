# Post Format Archetypes for Content Hub

## 1. digest_take
最新ニュースの要点を一言で翻訳する。
例: "このニュース、機能追加そのものより『運用コストが誰に乗るか』を見るほうが大事。"

## 2. practical_take
実務でどう使うか、どこから試すかを短く示す。
例: "新しいAIツールを試すとき、まず見るのは精度じゃなくてログの残しやすさ。改善が回るかはそこ。"

## 3. question
実務者の反応を引き出す問い。
例: "AI駆動開発で一番時間が溶けるの、実装よりレビュー待ちじゃない？"

## 4. contrarian
流行への少し逆張りな見方。
例: "AIエージェント時代でも、結局ボトルネックは『何を自動化しないか』の判断だと思う。"

## 5. japan_signal
日本市場の動きを、実務者向けに解釈する。
例: "海外ニュースより、日本の制度変更や業界の動きのほうが次の案件に効いたりする。"

## 6. career_angle
ニュースをキャリアや働き方の話に接続する。
例: "AIで作業時間が減るほど、何を捨てて何を残すかでキャリア差が出る。"
